(a)
When n > 0,
t(n) = (sum from j=0 to n-1) [t(j)] + n
= t(n-1) + (sum from j=0 to n-2) [t(j)] + n
= t(n-1) + t(n-1) - (n-1) + n
= 2t(n-1) + 1
